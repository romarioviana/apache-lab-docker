# apache-lab-docker

## Exemplo de uma imagem Docker com o conteúdo: 
* Ubuntu 20.04
* Apache 2
* Página index.html no servidor web apache
* Exposição da porta 80 do servidor web
* Incialização do servidor web na inicialização da imagem.
